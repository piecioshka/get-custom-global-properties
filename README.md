# get-custom-global-properties

:hammer: Get custom global properties

## Usage

In HTML file:

```html
<head>
    <!-- As first script! //-->
    <script src="src/main.js"></script>
</head>
```

⚠️ **WARNING**

To get best results please add as first script on the page.
